This demonstrates how the Open Finances page can be embedded on any website using an iframe.

<iframe src="https://yourdomain.com/open-finances/[USER_ID]" width="100%" height="600" frameborder="0" style="border: 1px solid #e5e7eb; border-radius: 8px;"> </iframe>
Note: This is a demonstration file. To see the actual embed in action, you need to:
  1. Start the Numera application with pnpm dev
  2. Configure Open Finances in Settings
  3. Replace USER_ID in the iframe src with your actual user ID
  4. Open this HTML file in your browser
